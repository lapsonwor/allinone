# css

## How to automatically resize the image to fit the "div" container?
- CSS
```css
max-width: 100%;
max-height: 100%;
```
- React
```javascript
{
    maxWidth: "100%",
    maxHeight: "100%",
}
```

## How to make the background image fit the whole page?
- CSS
```css
  position: fixed; 
  top: 0; 
  left: 0; 
  min-width: 100%;
  min-height: 100%;
```
- React
```javascript
{
    position: "fixed",
    top: "0",
    left: "0",
    minWidth: "100%",
    minHeight: "100%",
}
```
